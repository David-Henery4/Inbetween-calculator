# Inbetween calculator

This is a very small project that give you the middle value betwen 2 numbers. It was built because I find myself a lot of times, either having to use a calculator or having to work it out myself, when working on projects. 

It's a small tool that helps me when building projects, even if the gains are minimal. 😂
Give it a go it you want!

Although it was a very basic tool to build, I gave me the opportunity to use Github Pages for the deployment for the first time. I used vite so I had to use a github action in order to handle the build step before then deploying it.
